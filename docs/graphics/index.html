    图形处理

    文件格式转换(5分钟)

    • 图象处理-图象分析-文件-数据输入
    • 选择文件格式-选择 jpg 格式文件-点击转换
    • 对应目录生成 msi 格式文件
    下载jpg 格式示例文件，转到百度网盘 提取码：vbox

    点文件的输入1(10分钟)

    • 设置工作目录
    • 图形处理-输入编辑-新建工程
    • 新建点文件、线文件、区文件
    • 打开、关闭、编辑 点线区 文件
    • 新建点、复制点、删除点、移动点
    • 矩阵复制：点编辑-阵列复制点
    如果工作目录不存在，新建的点文件将无法打开、编辑．

    点文件的输入2(12分钟)

    • 空心字
    • 上下标
      • AL#-2#=O#-3 \(Al_2O_3\)
      • #+ 上标
      • #- 下标
      • #= 正常
    • 分子式
      • /1/2000/ \(\frac1{2000}\)
    • 缩放技巧
      • F5 放大
      • F6 中间
      • F7 缩小
